Module: nugget

Nugget Data Objects Nugget's provide business logic around one or more schema. The model validates data against the schema outside of the data connection. This makes it easy for the same model to support different backend DB's. Supported DB: MongoDB via Mongorita - Provides document/record storage Cayley via cayley.js - Provides relationships and complex indexing
